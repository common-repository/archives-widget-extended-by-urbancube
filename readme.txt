== Description ==

Widget to display archives with some options

== Installation ==

1. Upload `archives-widget-extended-by-urbancube` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Does it allows to upload larger files =

Yes and no.
It allows to upload xml zipped files, so the original file can be larger, but the total filesize allowed is the same.
